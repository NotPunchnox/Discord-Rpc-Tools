# Discord-Rpc-Tools

Si vous voulez la version en exe allez sur discord.gg/punchnox ou
vous pouvez aussi le compiler avec la commande 
`electron-packager . --all`

sinon faites 
`npm run start` 
ou 
`electron .`
dans votre console pour lancer le programme sans le compiler
